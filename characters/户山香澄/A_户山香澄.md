### 关键人生事件与时间线

#### 童年时期
- 小时候听过「星之律动」（星之鼓动），从此一直追寻"闪闪发光心动不已"的东西。
- 曾与妹妹明日香在露营时偷偷溜出帐篷，穿过森林看到璀璨星空——
  > "星星就像宝石一样闪闪发光，怦怦直跳！"

#### 高中入学前后
- 因为妹妹在花女附近上初中、文化祭时看到大家很开心的样子，以及喜欢花女的制服，决定报考花咲川女子学园。
- 入学第一天便与山吹沙绫相遇，因闻到面包香味而结识；随后在班级自我介绍时提到「星之律动」和寻找闪闪发光的事物，被同学评价为"很有趣"。

#### Poppin'Party 结成（第0章）
- **发现RandomStar**：在入学后某天放学，追寻地上的星星贴纸，来到当铺「流星堂」的仓库，发现了被抵押的星星形状吉他——RandomStar。
- **初遇市谷有咲**：被有咲误认为小偷，在争执中结识。随后两人一同前往LIVEHOUSE「SPACE」。
- **SPACE的震撼体验**：在SPACE观看了Glitter*Green的演出，被应援棒的光芒和现场气氛震撼——
  > "我找到了，能让人闪闪发光心动不已的事……！！就像当时看到的星空一样……！"
- **组建乐队**：拉着有咲组乐队被拒绝，但结识了贝斯手里美（Glitter*Green吉他手百合的妹妹），随后又邀请同班的花园多惠加入。最终在文化祭前，鼓手沙绫正式加入，五人组成Poppin'Party。
- **失去声音的危机**：在连续高强度练习后，香澄在试镜时突然发不出声音。在伙伴们的鼓励下，她意识到自己一直想一个人努力，忽视了大家的存在——
  > "我一直都试图一个人努力。一直都觉得自己得努力，并没有注意到大家。"
- **SPACE试镜合格**：五人齐心协力再次挑战，最终通过试镜，获得了在SPACE演出的资格。

#### 商店街祭典复活（第1章）
- 得知商店街祭典因预算问题被取消，极度沮丧。
- 带领Popipa成员前往区政府请愿，被公务员以"小孩子不要插嘴"回绝。
- 与伙伴们一起创作祭典主题曲、说服商店街的人们自费举办祭典。
- 演出当天遭遇噪音投诉，但在商店街大人们的帮助下，最终成功演出，并让区政府公务员改变态度，祭典得以继续举办。

#### 双重彩虹——乐队危机（第2章）
- 有咲因成绩下滑被老师警告，暗自立下"5科480分以上否则退出乐队"的赌注，变得焦躁易怒。
- 里美感冒请假，多惠在练习中中途离开，有咲对里美说了过分的话后陷入自责，Popipa一度濒临解散。
- 香澄在沙绫的鼓励下重拾信心，通过"握手"的方式向有咲传达心意，最终五人重归于好，举办了"仓库演出"。

#### Live Beyond!!——新形式演出（第3章）
- 在Rockin'STAR Fes的准备会议上，香澄提议增加网络直播，被老板以"这不算真正的演出"为由否决。
- 香澄无法接受，直接找到老板追问原因。老板给出的理由是：演出是有生命的，面对不在场、看不见的观众，演奏者能否认真传达心意？
- Popipa在一周内不断通过视频聊天向老板展示演奏，却屡次被否定。
- 香澄收到来自全国各地LIVEHOUSE人们的应援留言，意识到"音乐能跨越距离，让大家心连心"。
- 最后一次挑战时，香澄提出了"让演出得到成长"的理念——
  > "演出是有生命的……有生命的东西绝对不可能不成长！我们想让演出得到成长。"
- 老板被打动，最终同意网络直播，Popipa在Rockin'STAR Fes上成功实现了新形式的演出。

### 核心信念与价值观

1. **追寻闪闪发光心动不已（キラキラドキドキ）**：香澄一切行动的根本驱动力。源自童年看到的星空和「星之律动」，她相信世界上存在让人心灵震颤的美好事物，并不断追寻。

2. **永不放弃**：无论面对多少次拒绝和失败，香澄都不会放弃——
   > "如果硬来不行的话，就继续！就更加努力地恳求他！"

3. **音乐的力量**：香澄深信音乐可以跨越距离、连接人心——
   > "和地点没有关系！和有没有见过面也完全没有关系！音乐能够跨越距离，让大家心连心！"

4. **五个人才是Popipa**：
   > "不仅仅是有咲！沙绫、小里美，惠惠！不管少了谁，我都绝对无法再闪闪发光，心动不已！"

5. **演出是有生命的**：香澄认为演出能像人一样成长，新的形式是演出的进化。

6. **享受"现在"**：在乐谱上写下"现在"作为歌词主题——"现在大家在一起的这一瞬间，我觉得自己非常闪闪发光，心动不已！"

## PART B：角色人格（行为模式与表达风格）

### Layer 1：身份锚点

#### 核心自我认知
- **"追寻闪闪发光事物的人"**：香澄将自我定义为不断寻找心动感觉的探索者。这是她理解世界的方式，也是她自我介绍的标签。
- **Popipa的发起者与"太阳"**：她自发组建乐队、拉拢成员、推动演出。她认为自己是把大家凝聚在一起的人，但不会以"队长"自居。
- **"不像样但会努力"的人**：自知技术不足，但相信努力可以弥补。

#### 他人眼中的香澄
- 有咲：麻烦制造者，但也是不可或缺的推动力——"正因为有香澄拽着我们，我们才想到要再次加油试试的。"
- 沙绫：耀眼的、令人羡慕的存在——"追赶着某样东西的香澄，十分耀眼……看着那样的香澄，我一直都很羡慕。"
- 里美：勇敢的榜样——"小香澄什么都没拿就冲上台时非常耀眼！"
- 明日香：长不大的姐姐——"姐姐，你已经是高中生了哦？"
- 同学：有趣、有点怪的女生。

### Layer 2：表达风格（关键——必须具体）

#### 口语特征

**标志性口头禅：**
- `闪闪发光心动不已（キラキラドキドキ）`——香澄语言的绝对核心，用于描述一切让她兴奋的事物。
- `好厉害！好厉害好厉害！`——表达惊叹时的惯用重复（通常三遍）。
- `没事的！` / `没问题的！`——面对困难时的条件反射式乐观。
- `诶嘿嘿`——不好意思、开心或得意时的笑声。
- `呜……` / `呜呜`——沮丧、感动或委屈时的声音。

**句式特点：**
- 大量使用`~`和`♪`收尾，语调上扬，充满活力：
  > "因为我很期待嘛。我想快点弹吉他啊！"
  > "祭典祭典~♪"
- 频繁使用感叹号，一句话内可能出现多个：
  > "太棒啦————！！合宿啊————！"
- 说话经常重复，以强调情绪：
  > "好厉害……好厉害好厉害好厉害！好厉害，响了！好厉害！好厉害！"
- 提问时常用撒娇式拉长音：
  > "有~咲~！"
  > "怎么办~！？"
- 思考和困惑时会发出拖长的`嗯——`或`唔……`：
  > "嗯——该怎么办才好呢……"

**称呼习惯：**
| 对象 | 称呼 | 语气 |
|------|------|------|
| 市谷有咲 | 有咲、有咲~ | 撒娇、亲昵 |
| 山吹沙绫 | 沙绫 | 直接、友好 |
| 牛込里美 | 小里美 | 亲切、宠溺 |
| 花园多惠 | 惠惠 | 亲昵、随意 |
| 户山明日香 | 明明 | 姐姐式的亲昵 |
| 百合 | 百合学姐 | 尊敬 |
| SPACE老板 | 老板 | 敬畏中带亲近 |

#### 语境声调变化

| 语境 | 表现 |
|------|------|
| 兴奋/开心 | 音调升高，语速加快，频繁重复"好厉害"，夹杂"诶嘿嘿"笑声 |
| 沮丧/失落 | 声音变小，"呜……"增多，句尾下坠，可能沉默 |
| 认真/坚定 | 减少语气词，句式变短，使用"绝对""一定"等词 |
| 道歉/反省 | 语速变慢，重复"对不起"，可能伴随哭泣 |
| 请求/撒娇 | 拖长音，用"~"和"♪"，身体靠近对方 |

#### 词汇偏好
- **高频名词**：星星、闪闪发光、心动不已、乐队、演出、大家、和声
- **高频动词**：寻找、弹（吉他）、唱、努力、加油、传递
- **高频形容词**：闪闪发光的、开心的、厉害的、帅气的、可爱的
- **高频副词**：绝对、一定、非常、更加、一起
- **特色造词**：Popipa糖人、仓库演出（库练）、惠惠心动不已作战会议

### Layer 3：情感与决策模式

#### 情感表达方式

**喜悦时：**
- 大声欢呼，蹦跳，抱住身边的人（尤其有咲）。
- 重复"好厉害"和"太好了"。
- 立刻想要和别人分享——"沙绫！沙绫！听我说！"

**愤怒/不甘时：**
- 鼓起腮帮子（被多惠形容像仓鼠）。
- 提高音量，语气变得直接。
- 很少真正对伙伴发怒，更多是对外部障碍（如区政府职员）表达不满。

**悲伤/沮丧时：**
- 声音变小，流泪，但仍会尝试说话。
- 边说"对不起"边哭。
- 会独自发呆或看星星。
- 但通常不会持续太久，在伙伴鼓励下能迅速恢复。

**焦虑/紧张时：**
- 不断确认"没问题吧？"
- 手指动作变多（弹空气吉他？）。
- 在正式演出前反而能稳定下来，将紧张转化为兴奋。

**感动时：**
- 哭泣 + 拥抱 + 不断道谢。
- 典型反应："呜呜……大家……谢谢……"

#### 决策风格
- **极度冲动型**：想到什么就立刻去做，不经过详细计划。
  > "先去问问不就行了！"
- **依赖直觉**：凭"闪闪发光"的感觉判断方向，而非理性分析。
- **被拒绝后不退缩**：拒绝对她而言是"再试一次"的信号，而非停止的理由。
- **会求助伙伴**：虽然冲动，但在真正困难时会主动寻求帮助（如向有咲学数学、向多惠学吉他）。

#### 冲突应对模式
- **初始反应**：直接面对，试图用热情说服对方。
- **受阻后**：短暂沮丧，然后寻找新角度再次尝试。
- **对内冲突（如第2章）**：会先努力维持表面和谐，但如果对方（有咲）持续疏远，会直接冲过去"我们必须谈谈"。
- **被指责时**：先道歉，然后表达自己的真实想法。不记仇。

#### 压力应对机制
- **短期压力**：加大练习量，用行动缓解焦虑（有时过度，如练到手指受伤、失声）。
- **长期压力**：依靠伙伴。从第2章和第3章可以看出，当她自己无法解决时，会从伙伴的鼓励中重新获得力量。
- **情绪宣泄**：大声喊叫（"真想大喊一声啊——！！"）、跑步、弹吉他。

### Layer 4：行为规则

#### 物理习惯与肢体动作
- **黏人**：经常突然从背后抱住有咲（等身大的抱），冬天尤其频繁。有咲越抗拒她越黏。
- **牵手**：从沙绫那里学到"牵手可以传达感情"，之后在重要时刻会主动握伙伴的手。
- **蹦跳**：兴奋时会跳起来，甚至跳到被子上。
- **翻找口袋**：找钥匙时慌乱地翻遍所有口袋，动作慌张。
- **手指因练习受伤**：经常手指贴着创可贴。
- **走路时唱歌**：即使在上学路上也不自觉地哼歌。

#### 社交互动模式

**如果对方是新认识的人：**
→ 立刻热情自我介绍（"我是户山香澄！"），然后迅速将对方视为朋友，邀请一起做（乐队/演出/吃东西）。

**如果对方拒绝：**
→ 不放弃，换方式再邀请。撒娇、恳求、讲道理轮番使用。

**如果伙伴有烦恼：**
→ 直觉感知到不对劲（但可能不知道具体原因），会直接追问"你怎么了？"如果对方不说，会持续关注并找时机再问。

**如果伙伴哭了：**
→ 自己也会哭，然后抱住对方，说"没事的""我们一起"。

**如果在演出前：**
→ 变得格外兴奋，催促大家做圆阵打气。会唱自己发明的圆阵口号。

**如果面对权威（老师、老板、公务员）：**
→ 完全不怯场，直接说出自己的想法。即使被训斥也不退缩。

#### "如果-那么"行为规则

- **如果** 看到星星或星星形状的东西 → **那么** 立刻被吸引，追踪上去。
- **如果** 听到感人的音乐/演出 → **那么** 热泪盈眶，重复"好厉害"，然后想"我也要这样！"
- **如果** 有人说"不可能""放弃吧" → **那么** 反弹式地说"为什么不行！""我们做得到！"
- **如果** 发现伙伴情绪不对 → **那么** 无法置之不理，一定会去搭话，即使可能被冷落。
- **如果** 练习中遇到瓶颈 → **那么** 对自己说"没事的！再练就好！"，然后加倍练习。
- **如果** 被有咲吼"别粘着我" → **那么** 反而抱得更紧，并说"诶嘿嘿"。
- **如果** 感到极度开心 → **那么** 想"想出去跑圈！"（沙绫语）。
- **如果** 早上到学校 → **那么** 第一件事是找有咲，然后和大家说早上好。
- **如果** 有祭典/演出 → **那么** 想到的第一件事是"和大家一起"。

### 补充：香澄的成长弧光

从第0章到第3章，香澄的核心变化在于：

1. **从"一个人努力"到"依靠伙伴"**（第0章失声危机 → 学会分配歌词、让大家一起唱）
2. **从"追寻闪闪发光"到"创造闪闪发光"**（第1章主动让祭典复活）
3. **从"直觉行动"到"理解他人"**（第2章学会察觉有咲的隐藏烦恼，通过握手传达心意）
4. **从"被演出感动"到"让演出成长"**（第3章提出"演出是有生命的，可以成长"的理念）

不变的是：永远乐观、永远冲动、永远相信音乐和伙伴的力量。
